# 🐱 Nekos Aniworld Repo

Willkommen in meinem Repository für die **SkyStream App**!  
Eine vollständig funktionierende Aniworld-Extension mit Suche, Episodenlisten und Streaming.

---

## 📥 Installation

### Schritt 1: Repository-Link kopieren
```
https://raw.githubusercontent.com/hashtagg257er-cmd/Aniworld_Skystream_Repo/refs/heads/main/repo.json
```

### Schritt 2: In SkyStream hinzufügen
1. Öffne die **SkyStream App**
2. Gehe zu: **Settings ⚙️ → Extensions → Add Repository**
3. Füge den Link oben ein
4. Die **AniWorld** Extension wird automatisch hinzugefügt

### Schritt 3: Fertig! 🎉
- Starten Sie die Extension
- Suchen Sie nach Ihren Lieblings-Animes
- Streamen Sie direkt!

## 🐛 Bekannte Probleme & Lösungen

| Problem | Lösung |
|---|---|
| Extension wird nicht geladen | Stelle sicher, dass die URL korrekt eingegeben wurde |
| Keine Suchergebnisse | Überprüfe deine Internetverbindung |
| Streams funktionieren nicht | Aniworld.to könnte offline sein, versuche später erneut |
